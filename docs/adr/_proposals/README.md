# ADR proposals

This directory tracks Proposed ADRs drafted by `/architect`.

Files here are non-binding. They may be created from a user-supplied one-line decision or a user-supplied draft, but AI must not invent binding decisions or mark an ADR accepted.

Proposal files must use `docs/adr/_proposals/NNNN-<slug>.draft.md` and must not be written to `docs/adr/NNNN-<slug>.md`. A human reviewer accepts, rejects, or revises the proposal and then moves accepted records into `docs/adr/`.

Keeping this README allows the empty proposal directory to be tracked.
